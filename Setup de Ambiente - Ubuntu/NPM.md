### Instalação

1. Primeiro vamos atualizar a nossa lista de repositórios:

    ```bash
    sudo apt update
    ```

2. Instalar NPM:

    ```bash
    sudo apt install npm
    ```

3. Verificar versão:

    ```bash
    npm -v
    ```
